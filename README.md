A standalone example of building a custom AI agent using [pydantic-ai](https://github.com/pydantic/pydantic-ai) with Anthropic's Claude.

## Requirements

**You need an Anthropic API key!**

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

## Installation

```bash
cd cp_standalone_example
uv sync
```

## Usage

```bash
# Interactive mode
uv run cow-agent ""
```
